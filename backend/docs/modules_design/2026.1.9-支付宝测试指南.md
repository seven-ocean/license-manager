## 🧪 测试步骤

### 步骤1: 创建订单并支付

#### API请求
```bash
# 1. 获取套餐列表
curl -X GET "http://localhost:28888/api/cu/packages" \
  -H "Content-Type: application/json"

# 2. 计算价格
curl -X POST "http://localhost:28888/api/cu/orders/calculate" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "package_id": "basic",
    "license_count": 1
  }'

# 3. 创建付费订单
curl -X POST "http://localhost:28888/api/cu/orders" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "package_id": "basic",
    "license_count": 1,
    "payment_method": "alipay"
  }'
```

## 🔍 常见问题排查

### 1. 支付链接无法访问
- 检查 `gateway_url` 是否正确
- 确认网络能访问支付宝沙箱环境

### 2. 回调失败
- 检查 `notify_url` 是否可公网访问
- 查看服务日志中的回调处理信息

### 3. 签名验证失败
- 确认私钥格式正确（PKCS8）
- 检查公钥是否匹配

### 4. 订单状态未更新
- 检查回调URL配置
- 查看支付宝回调日志
