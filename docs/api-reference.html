YuYoung 授权管理系统 API 参考

按模块汇总主要接口与方法，便于联调与排障

通用

Base URL

后端服务：http://<host>:28888

  • GET /health — 服务健康检查
  • GET /swagger/*any — Swagger 文档入口
  • Static /files/** — 静态文件（发票等）下载

管理端 API（/api）

Admin

公开接口

  • POST /api/v1/login — 登录
  • POST /api/v1/activate — 许可证激活（客户端）
  • POST /api/v1/heartbeat — 许可证心跳（客户端）
  • POST /api/payment/alipay/callback — 支付宝回调
  • POST /api/v1/payment/alipay/callback — 支付宝回调（别名）
  • GET /api/public/invoices/download — 发票下载（凭 token）
  • POST /api/leads — 线索收集

需要认证

  • POST /api/v1/logout — 退出登录
  • POST /api/v1/auth/refresh — 刷新 Token

客户管理

  • GET /api/customers — 列表
  • GET /api/customers/:id — 详情
  • POST /api/customers — 新增
  • PUT /api/customers/:id — 更新
  • DELETE /api/customers/:id — 删除
  • PATCH /api/customers/:id/status — 启用/禁用

枚举

  • GET /api/enums — 全部枚举
  • GET /api/enums/:type — 指定类型

授权码

  • GET /api/v1/authorization-codes — 列表
  • POST /api/v1/authorization-codes — 新增
  • GET /api/v1/authorization-codes/:id — 详情
  • GET /api/v1/authorization-codes/:id/download — 下载文件
  • PUT /api/v1/authorization-codes/:id — 更新
  • PUT /api/v1/authorization-codes/:id/lock — 锁定/解锁
  • DELETE /api/v1/authorization-codes/:id — 删除
  • GET /api/v1/authorization-codes/:id/changes — 变更记录

许可证

  • GET /api/v1/licenses — 列表
  • GET /api/v1/licenses/:id — 详情
  • POST /api/v1/licenses — 新增
  • PUT /api/v1/licenses/:id/revoke — 吊销
  • GET /api/v1/licenses/:id/download — 下载许可证

统计与仪表盘

  • GET /api/v1/stats/overview — 概览
  • GET /api/v1/dashboard/authorization-trend — 授权趋势
  • GET /api/v1/dashboard/recent-authorizations — 最近授权

发票（管理员）

  • GET /api/v1/invoices — 列表
  • GET /api/v1/invoices/:id — 详情
  • GET /api/v1/invoices/summary — 汇总
  • POST /api/v1/invoices/:id/reject — 驳回
  • POST /api/v1/invoices/:id/issue — 开具
  • POST /api/v1/invoices/upload — 上传PDF

套餐

  • GET /api/packages — 列表
  • GET /api/packages/:id — 详情
  • POST /api/packages — 新增
  • PUT /api/packages/:id — 更新
  • DELETE /api/packages/:id — 删除
  • PUT /api/packages/:id/status — 上下架

线索（管理员）

  • GET /api/leads — 列表
  • GET /api/leads/summary — 汇总
  • GET /api/leads/:id — 详情
  • PUT /api/leads/:id — 更新
  • PUT /api/leads/:id/status — 更改状态
  • DELETE /api/leads/:id — 删除

管理员专用

  • GET /api/v1/admin/system/info — 系统信息

客户门户 API（/api/cu）

Customer

公开接口

  • POST /api/cu/register — 注册
  • POST /api/cu/login — 登录
  • POST /api/cu/send-login-sms — 登录短信
  • POST /api/cu/send-register-sms — 注册短信
  • POST /api/cu/forgot-password — 忘记密码
  • POST /api/cu/reset-password — 重置密码
  • GET /api/cu/packages — 套餐列表

需要认证

资料与账户

  • GET /api/cu/profile — 资料读取
  • PUT /api/cu/profile — 资料更新
  • POST /api/cu/profile/send-current-phone-sms — 旧手机短信
  • POST /api/cu/profile/send-new-phone-sms — 新手机短信
  • PUT /api/cu/profile/phone — 修改手机号
  • PUT /api/cu/profile/password — 修改密码

订单与支付

  • POST /api/cu/orders/calculate — 价格试算
  • POST /api/cu/orders — 下单
  • GET /api/cu/orders/:order_id — 订单详情
  • PUT /api/cu/orders/:order_id/cancel — 取消订单
  • POST /api/cu/orders/:order_id/pay — 继续支付
  • GET /api/cu/orders — 我的订单
  • GET /api/cu/orders/summary — 订单汇总
  • GET /api/cu/payment/:payment_no/status — 支付状态
  • GET /api/cu/payment/history — 支付历史

授权码与设备

  • POST /api/cu/authorization-codes/:codeId/share — 分享授权码
  • POST /api/cu/authorization-codes/product-activation-code — 产品激活码
  • GET /api/cu/authorization-codes — 我的授权码
  • GET /api/cu/authorization-codes/summary — 授权码汇总
  • GET /api/cu/devices — 设备列表
  • GET /api/cu/devices/summary — 设备汇总
  • DELETE /api/cu/devices/:id — 解绑设备

发票（用户侧）

  • POST /api/cu/invoices — 申请发票
  • GET /api/cu/invoices — 发票列表
  • GET /api/cu/invoices/:id — 发票详情
  • PUT /api/cu/invoices/:id — 更新发票信息
  • GET /api/cu/invoices/summary — 发票汇总
  • GET /api/cu/invoices/:id/download — 下载发票PDF

备注

  • 生产部署请通过 Nginx 前置代理；Compose 中 frontend 服务对外监听端口为 28080，后端为 28888。
  • 鉴权基于 Bearer Token；管理员接口需附加管理员角色。
  • 更多字段级说明请参考 /swagger 文档。
